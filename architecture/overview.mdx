## Core Interactions

The diagram below illustrates the key interactions among the network participants within the protocol.

Key Interactions Include:

<ol>
  <li>Issuers distribute or update PDAs for Owners.</li>
  <li>Verifiers request PDA details from Owners.</li>
  <li>Owners relay Proofs, which are snapshots of PDA data, to Verifiers.</li>
  <li>Verifiers compensate Issuers relative to the data leveraged.</li>
</ol>
