sidebarTitle: Fetch all PDAs issued by you
---

You can fetch all the PDAs issued by you using the `issuedPDAs` query. This query returns a list of all the PDAs issued by you.

```graphql
query issuedPDAs {
  issuedPDAs {
    id
    issuanceDate
    ownerHash
    dataAsset {
      claim
      organization {
        id
      }
      owner {
        id
      }
    }
    status
  }

  issuedPDAsCount
}
```
