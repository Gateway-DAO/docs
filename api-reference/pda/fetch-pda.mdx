sidebarTitle: Fetch a PDA
---

You can fetch the details of a PDA by using the `PDA` query. You require the PDA ID to fetch the details of a PDA.

```graphql
query PDA($id: String!) {
  PDA(id: $id) {
    id
    issuanceDate
    ownerHash
    issuerHash
    dataAsset {
      title
      dataModel {
        id
        createdAt
      }
      claim
      issuer {
        id
      }
      image
    }
    status
  }
}
```

<RequestExample>

```ts Node.js
const axios = require("axios");
let data = JSON.stringify({
  query: `YOUR_GRAPHQL_QUERY_HERE`,
  variables: {
    id: "YOUR_PDA_ID",
  },
});

let config = {
  method: "post",
  maxBodyLength: Infinity,
  url: "https://testnet.protocol.mygateway.xyz/graphql",
  headers: {
    "X-Api-Key": "<YOUR_API_KEY>",
    Authorization: "Bearer <YOUR_AUTHENTICATION_TOKEN>",
    "Content-Type": "application/json",
  },
  data: data,
};

axios
  .request(config)
  .then((response) => {
    console.log(JSON.stringify(response.data));
  })
  .catch((error) => {
    console.log(error);
  });
```

</RequestExample>
