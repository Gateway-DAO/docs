---
title: Create Wallet Nonce
description: Nonce required to be signed by the user wallet.
---

Whilst authenticating with Gateway Protocol, the wallet needs to sign a nonce. This nonce can be generate by using the `createWalletNonce` mutation.

| Parameter | Description                                            | Possible Values   |
| --------- | ------------------------------------------------------ | ----------------- |
| wallet    | The wallet address of the user.                        | String            |
| chain     | The chain you are interacting with. Default Value: EVM | EVM, SOLANA, POKT |

```graphql
mutation ($wallet: String!) {
  createWalletNonce(input: { wallet: $wallet, chain: SOL }) {
    message
  }
}
```

<RequestExample>

```ts Node.js
const axios = require("axios");

let data = JSON.stringify({
  query: `mutation($wallet: String!) {
  createWalletNonce(input: { wallet: $wallet
  chain: SOL
  }) {
    message
  }
}`,
  variables: { wallet: "SOL_WALLET_ADDRESS" },
});

let config = {
  method: "post",
  maxBodyLength: Infinity,
  url: "https://protocol.mygateway.xyz/graphql",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY",
    Authorization: "Bearer YOUR_AUTHENTICATION_TOKEN",
  },
  data: data,
};

axios
  .request(config)
  .then((response) => {
    console.log(JSON.stringify(response.data));
  })
  .catch((error) => {
    console.log(error);
  });
```
