You can fetch financial transactions using the `financialTransactions` query. You can specify the `identifier` to filter the transactions based on the user or organization.

Identifier can be one of the following:

- `USER`
- `ORGANIZATION`

```graphql
query financialTransactions($identifier: TransactionIdentifierInput!) {
  financialTransactions(identifier: $identifier) {
    id
    value
    type
    transactionId
    createdAt
    action
    memo
  }
}
```

<RequestExample>

```ts Node.js
const axios = require("axios");
let data = JSON.stringify({
  query: `query financialTransactions($identifier: TransactionIdentifierInput!) {
  financialTransactions(
    identifier: $identifier
  ) {
    id
    value
    type
    transactionId
    createdAt
    action
    memo
  }
}`,
  variables: {
    identifier: { type: "USER", id: "8ea4a126-c472-4671-a7e0-6cecc98953d0" },
  },
});

let config = {
  method: "post",
  maxBodyLength: Infinity,
  url: "https://protocol.mygateway.xyz/graphql",
  headers: {
    "X-Api-Key": "<YOUR_API_KEY>",
    Authorization: "Bearer <YOUR_AUTHENTICATION_TOKEN>",
    "Content-Type": "application/json",
  },
  data: data,
};

axios
  .request(config)
  .then((response) => {
    console.log(JSON.stringify(response.data));
  })
  .catch((error) => {
    console.log(error);
  });
```

</RequestExample>
