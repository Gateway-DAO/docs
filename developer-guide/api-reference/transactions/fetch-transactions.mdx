You can fetch transactions using the `transactions` query. You can specify the `filter` to filter the transactions based on the `from`, `to`, `cost`, and `action`.

```graphql
query transactions($filter: FilterTransactionsInput) {
  transactions(filter: $filter) {
    from {
      ... on Organization {
        id
        gatewayId
      }
      ... on User {
        id
        gatewayId
      }
    }
    to {
      ... on Organization {
        id
        gatewayId
      }
      ... on User {
        id
        gatewayId
      }
    }
    cost
    action
  }

  transactionsCount
}
```
