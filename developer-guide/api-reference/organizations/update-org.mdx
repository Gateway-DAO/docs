You can update the organization details using the `updateOrganization` mutation.

```graphql
mutation updateOrganization(
  $id: String!
  $username: String!
  $description: String!
) {
  updateOrganization(
    input: { id: $id, username: $username, description: $description }
  ) {
    id
    gatewayId
  }
}
```
