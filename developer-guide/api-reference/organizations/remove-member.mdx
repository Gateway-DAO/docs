You can remove a member from an organization using the `removeMemberFromOrganization` mutation. You can specify the `organization` and `user` of the organization and user respectively.

The organization can be identified by one of the following:

- `GATEWAY_ID`
- `ORG_ID`

The user can be identified by one of the following:

- `GATEWAY_ID`
- `EMAIL`
- `EVM`
- `SOLANA`
- `POKT`
- `USER_ID`

```graphql
mutation removeMemberFromOrganization {
  removeMemberFromOrganization(
    input: {
      organization: { type: GATEWAY_ID, value: "YOUR_ORG_GATEWAY_ID" }
      user: { type: GATEWAY_ID, value: "USER_GATEWAY_ID" }
    }
  )
}
```
