You can add a member to an organization using the `addMember` mutation. You can specify the `organization` and `user` of the organization and user respectively. The `role` is used to specify the role of the user in the organization.

The organization can be identified by one of the following:

- `GATEWAY_ID`
- `ORG_ID`

The user can be identified by one of the following:

- `GATEWAY_ID`
- `EMAIL`
- `EVM`
- `SOLANA`
- `POKT`
- `USER_ID`

Role can be one of the following, by default its `Member`:

- `Member`
- `Admin`
- `Owner`

```graphql
mutation addMemberToOrganization {
  addMemberToOrganization(
    input: {
      organization: { type: GATEWAY_ID, value: "YOUR_ORG_GATEWAY_ID" }
      user: { type: GATEWAY_ID, value: "USER_GATEWAY_ID" }
      role: "Member"
    }
  ) {
    id
  }
}
```
