You can create an organization using the `createOrganization` mutation. You can specify the `name` and `description` of the organization. The `username` is used to create a unique identifier for the organization.

```graphql
mutation createOrganization(
  $name: String!
  $username: String!
  $description: String!
) {
  createOrganization(
    input: { username: $username, name: $name, description: $description }
  ) {
    id
    gatewayId
    accesses {
      id
      role
      user {
        gatewayId
      }
    }
  }
}
```
