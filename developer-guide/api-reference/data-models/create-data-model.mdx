You can create a new data model specific to your requirements using the following mutation.

```graphql
mutation {
  createDataModel(
    input: {
      title: "Your Data Model Title"
      description: "A brief description of your data model."
      permissions: ALL
      tags: ["Tag1", "Tag2"]
      schema: {
        type: "object"
        default: {}
        title: "Your Data Model Title"
        required: ["field1", "field2", "field3"]
        properties: {
          field1: { type: "string", title: "Field 1" }
          field2: { type: "number", title: "Field 2", default: 0 }
          field3: { type: "boolean", title: "Field 3", default: false }
        }
      }
    }
  ) {
    id
    title
    schema
    allowedUsers {
      id
      gatewayId
    }
    allowedOrganizations {
      id
      gatewayId
    }
  }
}
```
