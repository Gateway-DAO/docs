You can reject a data request using the `rejectDataRequest` mutation. You will need the Data Request ID.

```graphql
mutation {
  rejectDataRequest(requestId: "YOUR_DATA_REQUEST_ID") {
    id
  }
}
```

<RequestExample>

```ts Node.js
const axios = require("axios");

let data = JSON.stringify({
  query: `mutation {
    rejectDataRequest(requestId: "YOUR_DATA_REQUEST_ID") {
      id
    }
  }`,
  variables: {},
});

let config = {
  method: "post",
  maxBodyLength: Infinity,
  url: "https://protocol.mygateway.xyz/graphql",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY",
    Authorization: "Bearer YOUR_BEARER_TOKEN",
  },
  data: data,
};

axios
  .request(config)
  .then((response) => {
    console.log(JSON.stringify(response.data));
  })
  .catch((error) => {
    console.log(error);
  });
```

</RequestExample>
