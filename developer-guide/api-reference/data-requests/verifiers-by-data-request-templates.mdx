This query returns the verifiers associated with a specific data request template.

```graphql
query {
  verifiersByDataRequestTemplate(id: "YOUR_DATA_REQUEST_TEMPLATE_ID") {
    verifier {
      ... on User {
        id
        gatewayId
      }
      ... on Organization {
        id
        gatewayId
      }
    }
    count
  }
}
```
