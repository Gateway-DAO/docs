The Gateway Protocol API (the "API") gives developers access to all aspects of the PDA. Developers can use the API to [Authenticate](/developer-guide/api-reference/auth/authenticate-account) users, define [Data Models](/developer-guide/api-reference/data-models/get-data-models), and essential [Create](/developer-guide/api-reference/data-assets/create-a-new-data-asset), [Read](/developer-guide/api-reference/data-assets/get-my-data-assets), [Update](/developer-guide/api-reference/data-assets/update-data-asset-by-id), and [Delete](/developer-guide/api-reference/data-assets/delete-data-asset-by-id) functionality.

The API is a RESTful API and can be accessed via the [Swagger documentation](https://dev.api.gateway.tech/swagger/index.html) and the documentation on this site directly.
