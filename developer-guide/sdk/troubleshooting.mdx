---
title: Troubleshooting
---

| Error                                                              | Solution                                                                                                                  |
| ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------- |
| `Error: Missing or incorrect API Key, check your developer portal` | Check if you are using the correct API Key and passing it correctly.                                                      |
| `Error: Cannot find module 'rpc-websockets/dist/lib/client'`       | Check your node version. Recommended Node v18. Delete `node_modules` and `package-lock.json` and run `npm install` again. |
