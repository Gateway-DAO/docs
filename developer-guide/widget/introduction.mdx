Before starting, note that Gateway Widget Issue can only be used for issue [PDAs](/core-concepts/personal-data-assets) being an organization. Make sure that you already have an Organization and [Data Model](/core-concepts/data-models).

<Warning>Only organizations can create widget sessions</Warning>

## What is Gateway Widget Issue?

Gateway Widget Issue is a tool that allows you to create a widget session for issue PDAs. This tool is useful for organizations that want to create a widget session for issue PDAs.
